# StickyLayout 侧边吸顶布局

StickyLayout 是一个侧边吸顶布局组件，用于创建具有固定侧边栏的页面布局。当页面滚动时，侧边栏会相对于父容器自动吸顶显示，同时内容区域会根据侧边栏宽度自动调整内边距

## 功能特性

- 🎯 **智能吸顶**: 左侧和右侧区域在滚动时自动吸顶显示（相对于外部容器）
- 📏 **自适应布局**: 内容区域根据侧边栏宽度自动调整padding，侧边栏区域高度也会自动计算
- 📱 **响应式**: 自动适应不同屏幕尺寸和滚动状态
- ⚡ **性能优化**: 使用 ResizeObserver 和滚动监听优化性能

## 注意事项

- 外部可滚动容器不要设置padding、border等属性，否则会影响内部尺寸计算
- 建议为侧边栏设置固定宽度，避免内容溢出
- 在嵌套滚动容器中使用时，需要正确设置 `scrollContainer` 属性

## API

### Attributes

| 属性名          | 说明                                                                | 类型                                | 默认值 |
| --------------- | ------------------------------------------------------------------- | ----------------------------------- | ------ |
| scrollContainer | 指定滚动容器，可以是CSS选择器字符串或HTMLElement对象。默认为 window | ^[string] \| ^[object]`HTMLElement` | —      |

### Slots

| 名称    | 说明                                                | 参数                 |
| ------- | --------------------------------------------------- | -------------------- |
| default | 主要内容区域，会根据左右侧边栏的宽度自动调整padding | —                    |
| left    | 左侧边栏区域                                        | `{ height: number }` |
| right   | 右侧边栏区域                                        | `{ height: number }` |
