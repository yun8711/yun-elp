# 总览

## 项目背景

Yun-Elp 组件库是一个基于 Vue 3 的企业级业务组件库，通过总结日常业务中常见需求，并参考其他优秀组件库，在 [Element Plus](https://element-plus.org/) 基础上进行二次封装，在保持UI一致的基础上，减少重复开发，专注于快速实现业务需求。

## 技术栈

- **框架**: Vue 3.5+
- **组件库**: Element Plus 2.10.2+
- **语言**: TypeScript 5+
- **CSS框架**：scss
- **构建工具**: Vite 5+
- **包管理器**: pnpm 9+
- **代码规范**: ESLint、Stylelint、prettier、Commitlint等
